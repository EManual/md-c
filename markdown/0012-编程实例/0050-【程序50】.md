题目：include 的应用练习　　　
程序源代码：
test.h 文件如下：
```  
define LAG >
define SMA <
define EQ ==
include "test.h" /*一个新文件50.c，包含test.h*/
include "stdio.h"
void main()
{ 
	int i=10;
	int j=20;
	if(i LAG j)
		printf("\40: %d larger than %d \n",i,j);
	else if(i EQ j)
		printf("\40: %d equal to %d \n",i,j);
	else if(i SMA j)
		printf("\40:%d smaller than %d \n",i,j);
	else
		printf("\40: No such value.\n");
}
```
